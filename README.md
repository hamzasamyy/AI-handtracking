# AI Hand Tracker

## Description
Real-time computer vision system that detects and counts fingers using Python, OpenCV, and MediaPipe.

## Technologies
- Python
- OpenCV (cv2)
- MediaPipe

## Features
- Real-time hand detection
- Finger counting
- Works under different conditions

## How to Run
```bash
pip install opencv-python mediapipe
python main.py
